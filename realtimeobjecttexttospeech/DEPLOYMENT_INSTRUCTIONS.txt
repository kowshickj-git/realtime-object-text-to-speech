‚ùì TROUBLESHOOTING:
-------------------

Problem: "Python is not installed"
Solution: Install Python from python.org and check "Add to PATH"

Problem: "No module named 'win32com'"
Solution: Run manually: pip install pywin32

Problem: No audio output
Solution: Check Windows volume, ensure speakers not muted

Problem: Camera not opening
Solution: Close other apps using webcam (Zoom, Teams, etc.)

Problem: Model download stuck
Solution: Check internet connection, wait patiently (990MB file)
